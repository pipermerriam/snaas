Snowman Morse Code for Humans

Converting text to snowman morse code has never been easier. And it just got easier!

  • Command Line
  • Python
                                    
$ pip install snorse
$ snorse "abc"
☃ ⛄⛄⛄   ⛄⛄⛄ ☃ ☃ ☃   ⛄⛄⛄ ☃ ⛄⛄⛄ ☃
$ desnorse "☃ ⛄⛄⛄   ⛄⛄⛄ ☃ ☃ ☃   ⛄⛄⛄ ☃ ⛄⛄⛄ ☃"
abc
                                    
                                
                                    
>>> import snorse
>>> print snorse.snorse("abc")
☃ ⛄⛄⛄   ⛄⛄⛄ ☃ ☃ ☃   ⛄⛄⛄ ☃ ⛄⛄⛄ ☃
>>> print snorse.desnorse("☃ ⛄⛄⛄   ⛄⛄⛄ ☃ ☃ ☃   ⛄⛄⛄ ☃ ⛄⛄⛄ ☃")
abc

Now you can Snorse/Desnorse at Web-Scale

With Snorse as a Service you get the convenience of snorse without none of the setup or configuration hassle. Snorsing has never been easier.

  • Command Line
  • Python
                                    
$ CURL -X POST "http://snorse-as-a-service.herokuapp.com/api/v1/snorse/" -d "abc"
☃ ⛄⛄⛄   ⛄⛄⛄ ☃ ☃ ☃   ⛄⛄⛄ ☃ ⛄⛄⛄ ☃
$ CURL -X POST "http://snorse-as-a-service.herokuapp.com/api/v1/desnorse/" -d "☃ ⛄⛄⛄   ⛄⛄⛄ ☃ ☃ ☃   ⛄⛄⛄ ☃ ⛄⛄⛄ ☃"
abc
                                    
                                
                                    
>>> import requests
>>> response = requests.post("http://snorse-as-a-service.herokuapp.com/api/v1/snorse/", 'abc')
>>> print response.content
☃ ⛄⛄⛄   ⛄⛄⛄ ☃ ☃ ☃   ⛄⛄⛄ ☃ ⛄⛄⛄ ☃
>>> print requests.post("http://snorse-as-a-service.herokuapp.com/api/v1/desnorse/", response.content)
abc

Happy Snorsing
